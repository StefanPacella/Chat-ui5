# Chat-ui5
An example of chat based on ui5

This is just the front-end and for run this project you need to have npm and ui5.


Usage.
Open your terminal and type this:

```
git clone https://github.com/StefanPacella/Chat-ui5.git
```
```
cd Chat-ui5
```
```
npm install
```
```
ui5 build  
```
```
ui5 serve 
```
